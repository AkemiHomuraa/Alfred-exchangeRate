# 汇率查询

😘

**需要注意的是由于接口问题,源币种和目标币种的种类不一样….**

使用`hlist dc`来查询支持的源币种

使用`hlist tc`来查询支持的目标币种



使用`hl 金额 源币种 目标币种`格式来查询汇率



ps:源币种 = 扣账币种       目标币种 = 交易币种
